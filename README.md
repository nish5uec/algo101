# algo101

This repo has implementations of basic algorithms.

## Added example: 3Sum in Java

See `src/ThreeSum.java` for a complete Java implementation of:

- Sorting + two-pointer traversal
- Duplicate handling for unique triplets
- A `main` method with sample input/output

## Added example: Reverse Words in Java

See `src/ReverseWords.java` for a Java implementation that:

- Trims leading/trailing spaces
- Collapses multiple spaces between words
- Reverses word order while preserving each word
